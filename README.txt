Учительская 2.0 — доска проекта

Что внутри:
- index.html — главная доска из 7 столбцов.
- styles.css — оформление сайта.
- script.js — поиск, подсказки, мини-диагностика.
- pages/ — отдельные страницы для каждой карточки доски.

Теперь каждая карточка на доске открывает свою отдельную страницу.
Всего создано страниц: 35.

Как открыть:
1. Распакуйте архив.
2. Откройте index.html в браузере.
3. Нажмите на любую карточку.
